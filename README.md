# 🔽 VECTOR COMBAT

Developed by **rainduckling**

**Vector Combat** is a minimalist, high-speed action arcade game where geometry meets adrenaline. Descend into an infinite abyss, harness the power of light, and shatter every crimson obstacle in your path.

## 🎮 Core Mechanics

In the abyss, speed is your only ally. Master the "Vector Flow" to survive:

- **The Descent:** Gravity is constant. Move across floating platforms to position yourself.

- **The Strike:** Use your high-frequency laser to clear threats from a distance.

- **The Dash:** Transform into a golden bolt of invincibility. Dash through enemies to destroy them instantly and reset your momentum.

- **The Impact:** Experience intense screen-shake and particle-burst feedback with every kill.

| **Action**             | **Input**            |
| ---------------------- | -------------------- |
| **Move Left / Right**  | `A` / `D`            |
| **Jump**               | `W`                  |
| **Fire Laser (Cyan)**  | `Left Mouse Button`  |
| **Vector Dash (Gold)** | `Right Mouse Button` |

## 🚀 Installation Instructions

1. **Download** the `VectorCombat.zip` file.

2. **Extract** the contents to a folder of your choice.

3. **Run** `VectorCombat.exe`.

4. Survive the fall.
